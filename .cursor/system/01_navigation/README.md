# Navigation Overview

> AI Agent: Use these files to find the correct URL and UI path for any HRcloud module.

## How HRcloud Navigation Works

HRcloud uses a **sidebar + top navigation** layout:
- **Left Sidebar**: Lists all available modules as icons (collapsed) or text (expanded)
- **Top Bar**: Shows current module name, search, notifications, profile avatar
- **Main Content Area**: Module-specific content
