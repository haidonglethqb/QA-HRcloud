## Overview

Workmates is HRcloud's **internal social communication platform** - an internal social network for company-wide communication. Kudos is the **employee recognition and rewards** system built into Workmates.

### Key Capabilities
- Company Feed: post updates, announcements, polls
- Channels: topic-based sub-feeds with configurable privacy
- Chat: direct messaging between employees
- Pages: internal wiki-style pages
- Surveys: gather employee feedback
- Kudos: send recognition with optional points/rewards
- Analytics: track engagement, kudos activity
- Digital Signage: broadcast content to screens
- Campaigns: targeted communication campaigns

## Key Features & Business Rules

### Company Feed
- All employees can view the Company Feed by default
- Admins can restrict who can **post** to the feed
- Posts can include: text, images, videos, GIFs, polls, tags
- Posts can be **scheduled** for future publishing
- Posts can be **pinned** to top of feed
- **Mandatory Read Announcements**: employees must acknowledge and/or sign; admins can check compliance
- **Categories** and **Channels** organize content
- **Auto-generated posts**: new hire announcements, birthdays, anniversaries (configurable)
- Post settings: censoring, GIPHY enable/disable, feelings, reactions (all configurable by admin)

### Channels
- Topic-based sub-feeds
- Privacy options: Public, Private, Secret
- Each channel has roles: Owner, Admin, Member
- Post approval can be required per channel
- Channel analytics available (permission-based)

### Kudos
- Two modes: **Without Points** (recognition only) and **With Points** (rewards)
- Only one kudos post per person per type at a time (typically)
- Custom badges and backgrounds can be created
- **Points Funding**: admin sets budget for who can give/receive points
- **Spending limits**: per employee limits on how many points they can give/receive
- **Points Approval**: can require manager approval before points are granted
- **Auto Rewards**: auto-give points on birthdays/anniversaries
- **Nomination**: nominate colleagues for special recognition
- Employees can **redeem** points for rewards (gift cards, etc.)
- Points can be **transferred** between coworkers
- **Bulk Import Kudos**: via CSV upload
- Analytics: track kudos given/received per person/team

### Surveys
- Create surveys with multiple question types
- Survey analytics available
- Surveys can be targeted to specific employee groups

### Ghostwriting
- Admin can post on behalf of another user

### Permissions
- Admins control who can: post, comment, create announcements, create channels, create pages
- "Notify Audience" permission is separately configurable
- Channel-level roles override global settings within that channel
- Analytics access is permission-controlled
