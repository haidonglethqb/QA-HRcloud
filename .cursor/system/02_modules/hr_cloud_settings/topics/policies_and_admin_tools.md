# HR Cloud Settings Policies and Admin Tools

Use this topic for policy settings, calendars, alerts, imports, custom fields, forms, and AI controls.

## Working Time Policy

- Defines working days in the week.
- Sets daily working hours.
- Feeds Time Off accrual and Time Clock calculations.
- Multiple policies can exist for different employee groups.

## Calendar Settings

- Company holidays can be excluded from working days.
- Calendar display settings are configured here.
- Google Calendar and Outlook integrations can connect here.

## Email Alerts

- Events can trigger notifications to admins, managers, or employees.
- Rules are configurable per module and per event.
- Recipients can be specific users, roles, or email addresses.

## Data Import

- Employee records can be imported by CSV.
- Time off balances and historical data can also be imported.
- System templates should be used to avoid mapping issues.

## Custom Fields

- Custom fields can be added to employee profiles.
- Field types include text, number, date, dropdown, checkbox, and multi-select.
- Custom fields can appear in reports.

## Forms Management

- E-form templates can be uploaded and managed.
- Merge fields support auto-population.
- Signature fields can be configured.
- Forms are reused in Onboard, Offboard, and Recruit flows.

## AI Settings

- AI features can be enabled or disabled.
- Data access boundaries for AI are configured here.
