## Overview

The Perform module manages **employee performance reviews** - including goal setting, review cycles, peer feedback, 360-degree reviews, and performance analytics.

### Key Capabilities
- Performance cycles (360-degree, top-down, peer, self-review)
- Goal tracking (individual and team goals)
- Manager check-ins / 1:1s
- Performance matrix (9-box grid)
- Analytics and talent trends
- AI-assisted review writing
- Integration with payroll for compensation decisions

## Key Features & Business Rules

### Performance Cycle Types
- **360-degree Review**: Self + manager + peers + direct reports all review
- **Top-Down Review**: Only manager reviews the employee
- **Self-Review**: Employee reviews themselves
- **Peer Review**: Peers selected by employee or set by admin
- Cycles can combine multiple review types

### Review Cycle Status Flow
1. **Draft**: Cycle created, not yet live
2. **Active**: Employees and managers can complete reviews
3. **Closed**: No more inputs accepted
4. **Completed**: Results visible, analytics available

### Review Sections
Each review form can include:
- **Competencies**: scored on rating scale
- **Goals**: mark progress on individual goals
- **Open-ended questions**: free text responses
- **Ratings**: overall rating (configurable scale: 1-5, 1-10, custom labels)
- **Signatures**: require sign-off on review by both parties

### Sharing Reviews
- Admin controls whether employee can see their review results
- Manager can share review with employee
- Sharing can be set as automatic after cycle closes

### Goals
- Goals can be: personal, team, or company-wide
- Status options: On Track, At Risk, Behind, Completed
- Goals can be aligned to company objectives (OKR-style)
- Goals carry over between cycles if not completed
- Progress can be updated anytime

### Performance Matrix (9-Box)
- Maps employees by: Performance Rating (Y axis) vs. Potential (X axis)
- 9 categories (from "Low Performer" to "Star Performer")
- Admin/HR can view and export matrix results

### AI Review Assist
- Suggests review text based on goal completion and competency ratings
- Helps managers write more consistent, objective reviews

### Permissions
- HR Admin: full access to all reviews and analytics
- Manager: see/complete reviews for their direct reports
- Employee: see only their own review (when shared by admin/manager)
- Peer: see only the review they were asked to complete
