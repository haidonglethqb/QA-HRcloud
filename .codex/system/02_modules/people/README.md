Read this module for employee profiles, employment data, documents, org chart, and core HRIS behavior.

## Dependencies

### Depends On
- **Authentication** — must login before accessing People module.
- **Navigation** — sidebar or direct URL `/people`.

### Used By
- **Onboard** — creates prehire employees that trigger onboarding workflows.
- **Offboard** — terminates employees via employee profile actions.
- **Perform** — employees referenced in performance cycles.
- **Time Off** — employees submit time off requests.
- **Time Clock** — employees clock in/out.
- **Assets** — assets assigned to employees.
