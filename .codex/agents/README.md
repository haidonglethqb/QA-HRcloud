# Codex Agents

These files are migrated from `.cursor/agents` for Codex use. Codex does not need a Cursor tool wrapper; read the matching agent file when the task calls for a specialist role.

- `requirement-analysis.agent.md` - requirement and Jira story analysis.
- `story-loophole-finder.agent.md` - find gaps, contradictions, missing edge cases, and unclear acceptance criteria.
- `tester.agent.md` - QA test design and validation planning.
- `mcp-manager.agent.md` - Atlassian MCP setup and troubleshooting.
- `planner.agent.md` - planning and task breakdown.
- `project-manager.agent.md` - project coordination and status summaries.
- `researcher.agent.md` - research and context gathering.
- `ui-ux-designer.agent.md` - UI/UX review and design feedback.
- `docs-manager.agent.md` - documentation maintenance.
- `git-manager.agent.md` - Git workflow guidance.
- `journal-writer.agent.md` - notes and progress journals.
